# Product

本层描述 Goal Diffusion 的产品/方法论定位。

## Owns

- Goal Diffusion 服务谁。
- 适用场景和非目标。
- 面向高智能 agent 的默认姿态。
- 用户为什么需要长期目标、滚动实施和证据收口。

## Must Not Own

- CLI 具体实现。
- Goal Pack 当前状态。
- checker 规则细节。
- 未采纳的迁移计划。

## 当前定位

Goal Diffusion 是面向高智能 agent 的长期 AI coding 方法论。它默认相信 agent 能自主实施，但要求目标授权、边界、证据检查点和完成收口足够清楚。

默认不是弱模型防御式流程，而是：

```text
strong-agent optimistic workflow
```
